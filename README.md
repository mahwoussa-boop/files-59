# 📄 محرر PDF الذكي — Smart PDF Editor

تطبيق **Streamlit** لتحرير واستبدال النصوص داخل ملفات PDF بطريقة أكثر ذكاءً. يدعم التطبيق الآن **البحث عن كلمة أو عبارة واستبدالها مباشرة**، مع الحفاظ قدر الإمكان على **الموضع واللون والحجم والخط التقريبي**، كما يدعم **OCR احتياطي** للصفحات الصورية، بالإضافة إلى **مساعدة اختيارية بالذكاء الاصطناعي** لاختيار أقرب تطابق عند فشل المطابقة الحرفية.

---

## التشغيل السريع

```bash
pip install -r requirements.txt
streamlit run app.py
```

> لتفعيل OCR محليًا على الخادم أو الجهاز، يلزم وجود `tesseract-ocr` ويفضل تثبيت حزم اللغات المطلوبة مثل العربية والإنجليزية.

## الميزات الحالية

| الميزة | الوصف |
|---|---|
| **استبدال ذكي** | اكتب الكلمة الأصلية والكلمة البديلة وسيحاول التطبيق العثور على أفضل تطابق داخل الصفحة الحالية |
| **تحرير يدوي** | اختيار عنصر نصي أو كلمة من القائمة وتعديلها مباشرة |
| **وضع الكلمات** | مناسب لتعديل كلمة منفردة |
| **وضع العناصر النصية** | مناسب لتعديل عبارة أو سطر أو مقطع كامل |
| **OCR احتياطي** | يستخدم التعرف الضوئي على الحروف عندما لا توجد طبقة نص أصلية داخل الصفحة |
| **مساعدة AI اختيارية** | تستخدم نموذجًا ذكيًا لتحديد أقرب عنصر نصي إذا لم يجد التطبيق تطابقًا مباشرًا |
| **دعم العربية والإنجليزية** | مع مطابقة أكثر مرونة عبر تطبيع النصوص |
| **تراجع عن آخر تعديل** | لحماية سير العمل أثناء التجربة |
| **تنزيل الملف المعدل** | تصدير PDF بعد التعديلات مباشرة |

## طريقة الاستخدام

ابدأ برفع ملف PDF، ثم اختر الصفحة المطلوبة. بعد ذلك يمكنك استخدام أحد المسارين:

| المسار | متى يُستخدم |
|---|---|
| **الاستبدال الذكي** | عندما تريد كتابة الكلمة الحالية والكلمة الجديدة ليقوم التطبيق بالبحث والاستبدال تلقائيًا |
| **التحرير اليدوي** | عندما تريد اختيار عنصر محدد من العناصر المستخرجة وتحريره بنفسك |

إذا لم تظهر نصوص قابلة للتحرير، فعّل خيار **OCR** من الشريط الجانبي. وإذا كان الملف يحتوي على اختلافات بسيطة في الكتابة أو المسافات أو الرموز، يمكنك تفعيل **المساعدة الذكية** لتحسين اختيار أقرب تطابق.

## ملاحظات مهمة

| الحالة | التوضيح |
|---|---|
| **ملف PDF نصي** | يعطي أفضل نتيجة لأن التطبيق يقرأ النصوص الأصلية مباشرة |
| **ملف PDF صوري** | يعمل عبر OCR، لكن الحفاظ على الخط الأصلي يكون تقريبيًا أكثر من الملفات النصية |
| **الخطوط المضمنة** | قد لا يمكن إعادة استخدامها حرفيًا في كل ملفات PDF، لذلك يستخدم التطبيق خريطة بدائل ذكية |
| **الذكاء الاصطناعي** | وظيفة اختيارية لتحسين المطابقة، وليست شرطًا لتشغيل التطبيق الأساسي |

## النتيجة

الهدف من هذه النسخة هو أن يشعر المستخدم أن التطبيق **يفهم النص الذي يريد استبداله**، وليس مجرد قائمة جامدة للعناصر. لذلك تم توسيع المحرر ليشمل البحث الذكي، والاستبدال المباشر، وOCR، والمساعدة الاختيارية بالذكاء الاصطناعي.
